# ACS (Agnostic Config Suites)

Satu routing engine. Satu installer. Lima agen AI coding disatukan dalam satu config stack.

## Apa itu ACS?

ACS adalah lapisan konfigurasi dan manajemen terpadu untuk perangkat AI coding:

- **9router** — Routing multi-akun, kompresi, fallback
- **Claude Code** — Perencanaan dan review interaktif
- **Hermes Agent** — Eksekusi otonom dan skill factory
- **OMP** — Harness coding presisi (LSP, DAP, hash edits)
- **OpenCode** — TUI interaktif sekunder
- **kiro-cli** — TUI interaktif AWS
